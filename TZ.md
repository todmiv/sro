# Техническое задание (ТЗ)
## Веб-приложение "НОСО - Вступление в СРО"

**Версия:** 1.0.0  
**Дата:** 05.12.2025  
**Заказчик:** НОСО (Нижегородское объединение строительных организаций)

## 1. Общие сведения

### 1.1. Назначение проекта
Создание современного веб-приложения для автоматизации процесса вступления строительных организаций в саморегулируемую организацию (СРО) НОСО в Нижнем Новгороде.

### 1.2. Цели проекта
- Предоставить удобный онлайн-инструмент для подачи заявок на вступление в СРО
- Автоматизировать расчет стоимости членских взносов и компенсационных фондов
- Увеличить конверсию заявок за счет улучшения пользовательского опыта
- Создать современный имидж организации в цифровом пространстве

### 1.3. Целевая аудитория
- Руководители строительных компаний
- Индивидуальные предприниматели в сфере строительства
- Специалисты по работе с документацией

## 2. Функциональные требования

### 2.1. Основные функции

#### 2.1.1. Калькулятор взносов СРО
- **Описание:** Интерактивный калькулятор для расчета стоимости членства
- **Функционал:**
  - Выбор уровня ответственности (до 90 млн ₽, до 500 млн ₽, до 3 млрд ₽, без ограничений)
  - Автоматический расчет членского взноса (руб/месяц)
  - Автоматический расчет компенсационного фонда
  - Динамическое обновление расчетов при изменении параметров

#### 2.1.2. Форма подачи заявки
- **Описание:** Онлайн-форма для сбора контактных данных и информации об организации
- **Поля формы:**
  - ИНН организации (обязательное, текст)
  - Контактное лицо (обязательное, текст)
  - Телефон (обязательное, телефон)
  - Email (обязательное, email)
- **Валидация:**
  - Проверка формата ИНН (10 или 12 цифр)
  - Проверка формата email
  - Проверка формата телефона (+7 XXX XXX XX XX)
  - Все поля обязательны для заполнения

#### 2.1.3. Информационные разделы
- **Преимущества НОСО:** Отображение ключевых преимуществ организации
- **Процесс вступления:** Пошаговое описание процедуры вступления
- **Необходимые документы:** Список требуемых документов
- **Шаблоны документов:** Возможность скачивания шаблонов

#### 2.1.4. Контактная информация
- Отображение контактных данных организации
- Телефон: +7 (831) 123-45-67
- Email: info@noso.ru
- Адрес: ул. Большая Покровская, д.15, оф.7, Нижний Новгород

### 2.2. Пользовательские сценарии

#### Сценарий 1: Расчет стоимости членства
1. Пользователь заходит на главную страницу
2. Выбирает уровень ответственности в калькуляторе
3. Система автоматически рассчитывает и отображает стоимость
4. Пользователь может изменить уровень и увидеть новый расчет

#### Сценарий 2: Подача заявки
1. Пользователь заполняет форму с контактными данными
2. Система валидирует введенные данные
3. При успешной валидации отображается подтверждение
4. Данные сохраняются для дальнейшей обработки

#### Сценарий 3: Получение информации
1. Пользователь просматривает преимущества организации
2. Изучает процесс вступления
3. Знакомится со списком необходимых документов
4. Скачивает шаблоны документов
